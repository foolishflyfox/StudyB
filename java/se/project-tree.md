# Java 项目结构

## 通常的项目结构

```
└── project-name
    └── src
        ├── main
        │   ├── java
        │   └── resources
        └── test
            └── java
```
- `project-name/src/main/java` 放置项目 java 源码；
- `project_name/src/main/resources` 放置项目静态资源和配置文件；
- `project_name/src/test/java` 放置项目测试用例代码；

其中，`/src/main/java` 目录下存放 Java 源代码。
